Run file battleShip.py in Python IDLE or PyCharm or in some other python editor. Version Python 2.7.12. is used. Package PyQt4 needs to be installed. First you need to set your layout. Select one of radio buttons and the ship will appear. Move your ship with W key (up), D key (right), S key (down) and A key (left). With H key you determine horizontal / vertical position. With Return / Enter key you confirm position. When you set your layout button press "Start game" button and play the game.
